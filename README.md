# AI Career Guidance - Enhanced Frontend (Profile + Dashboard)

Features
- React + Vite frontend
- Tailwind CSS for styling
- React Router for separate `/profile` and `/dashboard` pages
- Tag inputs for Skills and Interests
- Skill recommendation logic showing matching and missing skills
- Skill-gap bar chart (Chart.js via react-chartjs-2)
- LocalStorage persistence (no backend)

Run locally
1. `cd ai-career-frontend-enhanced`
2. `npm install`
3. `npm run dev`
4. Open the URL shown by Vite (usually http://localhost:5173)

Notes
- This is frontend-only. All data stays in your browser's localStorage.
- Project is ready for deployment to Vercel/Netlify after `npm run build`.
